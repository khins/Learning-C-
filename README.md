# C++ Fundamentals

This project is a small learning path for beginner C++ practice.

## Lesson Order

1. `main.cpp`
2. `lesson02_variables.cpp`
3. `lesson03_input.cpp`
4. `lesson04_if_statements.cpp`
5. `lesson05_loops.cpp`
6. `lesson06_functions.cpp`

Open one lesson file at a time and run that file as the active editor tab.

## Running In VS Code

The current VS Code workspace has two launch options:

- `Run active C++ file`
- `Debug active C++ file`

### See Program Output

1. Open the lesson file you want to run.
2. Press `Ctrl+Shift+D`.
3. Choose `Run active C++ file`.
4. Press the green play button, or press `F5`.

VS Code will build the active `.cpp` file and open a separate console window to show the output.

### Debug Step By Step

1. Open the lesson file you want to debug.
2. Click in the left margin to add a breakpoint.
3. Press `F5`.
4. Choose `Debug active C++ file`.

Useful debug keys:

- `F5` continue
- `F10` step over
- `F11` step into
- `Shift+F5` stop debugging

## Notes

- These lessons are built with `g++`.
- Programs that use `cin` will ask for input in the external console window.
- Built `.exe` files are ignored by git and do not need to be committed.
